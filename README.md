# 🔐 Kalivira

Kalivira is a privacy-first password management system built using Spring Boot and modern security practices.

##  Features

- Secure password storage
- AES Encryption
- PBKDF2 Password Hashing
- JWT Authentication
- Role Based Access
- Password Generator
- Password Strength Checker
- Secure Notes
- Audit Logs

---

##  Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- Maven

### Database
- MySQL

### Frontend
- React
